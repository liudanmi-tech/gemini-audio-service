### 方法 1: 通过 Finder 打开（推荐）

2. **导航到项目文件夹**：

   - 或者直接搜索：`WorkSurvivalGuide.xcodeproj`

3. **找到 `WorkSurvivalGuide.xcodeproj` 文件**
   - 应该是一个蓝色的 Xcode 项目图标

4. **双击打开**
   - 会启动 Xcode 并打开项目

5. **等待 Xcode 加载完成**
   - 底部状态栏会显示 "Indexing..." 或 "Ready"
   - 项目导航器应该显示蓝色项目图标

### 方法 2: 通过 Xcode 打开

1. **打开 Xcode**

2. **菜单栏**：`File` → `Open...`
   - 或按 `Cmd + O`

3. **导航到项目文件**：
   - 路径：`桌面/AI军师/gemini-audio-service/Models.swift/WorkSurvivalGuide/`
   - 选择 `WorkSurvivalGuide.xcodeproj`
   - 点击 "Open"

4. **等待 Xcode 加载完成**

---

### 方法 3: 直接拖拽打开

1. **打开 Finder**

2. **找到 `WorkSurvivalGuide.xcodeproj` 文件**

3. **拖拽到 Xcode 图标**
   - 在 Dock 中找到 Xcode 图标
   - 将 `.xcodeproj` 文件拖到 Xcode 图标上

4. **Xcode 会自动打开项目**

3. **Xcode 版本问题**
   - 可能需要更新 Xcode
