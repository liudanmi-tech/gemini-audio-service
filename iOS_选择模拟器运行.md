# iOS 选择模拟器运行

## 🎯 当前错误

**错误信息**：`A build only device cannot be used to run this target.`

**原因**：当前选择的是 "Any iOS Device" 或其他不能运行的设备，需要选择模拟器。

---

## ✅ 解决方案：选择模拟器

### 方法 1: 通过菜单栏选择（推荐）

1. **点击 "OK" 关闭错误提示**

2. **菜单栏**：`Product` → `Destination`
   - 会显示可用的设备列表

3. **选择模拟器**：
   - 在列表中找到 `iPhone 14 (iOS 18.6)` 或类似的模拟器
   - 点击选择

4. **然后运行项目**：
   - 菜单栏：`Product` → `Run`
   - 或按 `Cmd + R`

---

### 方法 2: 通过顶部工具栏选择

1. **点击 "OK" 关闭错误提示**

2. **查看 Xcode 顶部工具栏**
   - 在运行按钮旁边，应该有一个设备选择器
   - 可能显示：`Any iOS Device` 或其他

3. **点击设备选择器**
   - 会弹出设备列表

4. **选择模拟器**：
   - 在列表中找到 `iPhone 14 (iOS 18.6)` 或类似的模拟器
   - 点击选择

5. **然后运行项目**：
   - 点击运行按钮（▶）
   - 或按 `Cmd + R`

---

### 方法 3: 如果看不到设备选择器

1. **显示工具栏**：
   - 菜单栏：`View` → `Show Toolbar`
   - 或按 `Option + Cmd + T`

2. **然后按照方法 2 操作**

---

## 🆘 如果 Destination 列表中没有模拟器

### 可能的原因：

1. **模拟器没有创建**
   - 需要先创建模拟器（之前已经创建过了）

2. **模拟器列表没有刷新**
   - 尝试重新打开 Devices and Simulators
   - 或重启 Xcode

### 解决：

1. **打开 Devices and Simulators**：
   - 菜单栏：`Window` → `Devices and Simulators`
   - 或按 `Shift + Cmd + 2`

2. **切换到 Simulators 标签**

3. **确认模拟器存在**：
   - 应该能看到 `iPhone 14 (iOS 18.6)`

4. **如果存在，回到 Xcode**：
   - 菜单栏：`Product` → `Destination`
   - 应该能看到模拟器了
