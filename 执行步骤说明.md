# 执行步骤说明

## 📍 在哪里执行这些命令

**在阿里云控制台的"命令助手"中执行**

## 🎯 具体操作步骤

### 1. 打开阿里云控制台
- 登录 https://ecs.console.aliyun.com
- 或者登录阿里云控制台，找到"轻量应用服务器"

### 2. 找到你的服务器
- 在服务器列表中，找到 IP 为 `47.79.254.213` 的服务器
- 点击进入服务器详情页

### 3. 打开"命令助手"
- 在服务器详情页的顶部菜单栏，找到 **"命令助手"** 标签
- 点击"命令助手"

### 4. 执行命令
- 点击 **"执行命令"** 按钮
- 在弹出的对话框中：
  - **命令类型**：选择 "Shell"
  - **命令来源**：选择 "输入命令内容"
  - **命令名称**：可以填写 "检查代码" 或任意名称
  - **命令内容**：粘贴下面的命令

### 5. 执行并查看结果
- 点击"确定"或"执行"
- 等待执行完成
- 查看执行结果

## 📋 要执行的命令（逐个执行）

命令助手通常在：
- 服务器详情页 → 顶部标签栏 → **"命令助手"**
- 或者：服务器列表 → 点击服务器 → 左侧菜单 → **"命令助手"**
