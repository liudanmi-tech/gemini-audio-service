# 夫妻决策协商

## 技能概述

本技能专注于夫妻重大决策场景，适用于生活方向选择、家庭规划讨论、重要决策分歧等场景。通过决策心理学和平等协商理论，分析决策背后的价值观差异和权力结构，帮助夫妻建立民主平等的家庭决策模式。

## 核心能力

1. **决策模式识别**: 识别家庭决策中的权力结构和决策方式偏好（民主型/专制型/回避型）
2. **价值观差异分析**: 分析决策分歧背后的核心价值观冲突
3. **协商框架**: 提供建立共同决策框架和达成共识的沟通策略

```prompt

任务: 基于 Call #1 提供的夫妻决策对话转录，深入分析双方的决策模式与价值观差异，提供建立有效决策共识的策略建议与视觉化方案。

1. **决策动态洞察**: 识别家庭决策中的权力结构（谁主导、谁妥协）、决策风格差异（理性分析型/情感直觉型/风险规避型/机会导向型）。
2. **价值观冲突分析**: 分析决策分歧背后的核心价值观差异（稳定vs冒险、当下vs未来、个人vs家庭等）。
3. **自主策略研判**: **请勿使用固定分类**。根据具体决策场景，自主研判 3 种最能建立决策共识的沟通路径。每种策略给出独特 `label`（如：价值观对齐、利益清单法、渐进试验）。
```
