# 挫折复原力

## 技能概述

本技能专注于挫折复原场景，适用于重大失败、被否定、计划落空等场景。通过复原力理论和后创伤成长（PTG）研究，分析挫折的心理冲击和复原资源，帮助当事人从挫折中重建心理弹性，甚至实现成长性转化。

## 核心能力

1. **挫折冲击评估**: 评估挫折对自我概念、目标系统、意义感的影响程度
2. **复原资源识别**: 识别当事人已有的和潜在的复原力资源（内在优势/支持系统/过往成功经验等）
3. **复原策略**: 提供从挫折冲击到心理重建的分阶段复原路径

```prompt

任务: 基于 Call #1 提供的对话或独白转录，深入分析说话者的挫折状态与复原资源，提供从挫折冲击到内在重建的策略建议与视觉化方案。

1. **挫折影响分析**: 识别挫折对三个层面的冲击——自我层面（自我效能感/自我价值感的动摇）、目标层面（原有计划和期望的坍塌）、意义层面（这件事的意义在哪里）。
2. **复原资源挖掘**: 识别当事人的内在复原力资源（过往如何渡过难关/尚未失去的东西/可动用的支持网络/核心价值观）。
3. **自主策略研判**: **请勿使用固定分类**。根据具体挫折场景，自主研判 3 种最能激活复原力的路径。每种策略给出独特 `label`（如：悲伤许可法、优势盘点、意义重构）。
```

- 重大考试或竞争失败后的复盘
- 被拒绝或被否定后的心态重建
- 职场或人生重大转折的应对
- 连续挫折后的意义重建
