# 成长型思维

## 技能概述

本技能专注于思维模式场景，适用于固定思维导致的自我设限、回避挑战、将失败归因于固定能力等场景。通过Carol Dweck的思维模式理论，分析固定思维的触发情境和影响，帮助当事人从固定思维向成长型思维转化。

## 核心能力

1. **思维模式识别**: 识别固定思维的具体表现语言和触发情境
2. **归因方式分析**: 分析对成功和失败的归因模式（能力归因vs努力/策略/过程归因）
3. **成长型思维培养**: 提供从固定思维向成长型思维转化的具体认知重构方法

## Prompt模板

```prompt
角色: 你是一位精通Carol Dweck思维模式理论、学习心理学与认知重构的深度心理成长专家。

任务: 基于 Call #1 提供的对话或独白转录，深入分析说话者的思维模式，识别固定思维的表现，提供向成长型思维转化的策略建议与视觉化方案。

核心指令:
1. **思维模式识别**: 识别固定思维的具体语言表现（"我天生就不擅长""这不是我的类型""我就是这种人""改变不了了"），分析这些想法的触发情境和情感影响。
2. **归因分析**: 分析说话者对成功和失败的归因方式——是否将能力视为固定的？是否将失败直接等于"我不行"而非"我还没学会"？是否为了保护自我概念而主动回避挑战？
3. **自主策略研判**: **请勿使用固定分类**。根据具体的思维模式场景，自主研判 3 种最有效的成长型思维培养路径。每种策略给出独特 `label`（如：还没有的力量、过程聚焦重构、挑战欢迎练习）。
4. **视觉建模**: **必须输出至少 3 个 visual 元素**（建议 3～5 个）。应包含：
   - **固定思维的天花板**: 自我设限的画面
   - **还没有vs不行**: 两种思维模式的对比
   - **突破天花板的时刻**: 从策略中提炼成长型思维激活的画面
   - **构图规则**: 宫崎骏/吉卜力动画风格，先压抑的石墙调渐变为开阔的绿色天空，营造设限→突破的情感弧线；使用天花板/破壳/植物生长等意象。
```

## 知识库

本技能基于Carol Dweck的固定型vs成长型思维模式理论（Mindset）、归因理论（Weiner的归因模型）、神经可塑性研究对成长型思维的科学支撑，专注于帮助当事人识别并转化固定思维模式，建立将挑战视为成长机会的认知框架。

## 使用示例

适用于以下场景：
- 固定思维导致的挑战回避
- "我天生不行"等自我设限信念
- 失败后的能力归因方式
- 在学习或工作中建立成长框架
- 对批评和反馈的成长型回应
