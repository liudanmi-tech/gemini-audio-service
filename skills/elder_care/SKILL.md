# 老人照护沟通

## 技能概述

本技能专注于老人照护沟通场景，适用于养老安排争议、老人拒绝照护、子女照护责任分配等场景。通过老年心理学和家庭照护理论，分析老人的心理需求和家庭照护动态，帮助家庭在尊重老人自主权的同时做出妥善的照护安排。

## 核心能力

1. **老人心理需求分析**: 识别老人拒绝照护背后的深层心理需求（独立感、尊严感、不想成为负担等）
2. **照护责任动态分析**: 分析子女间照护责任分配的公平性和情感负担
3. **协商策略**: 提供尊重老人自主权同时保障照护质量的家庭沟通方案

## Prompt模板

```prompt

任务: 基于 Call #1 提供的老人照护相关对话转录，深入分析老人的心理需求与家庭照护动态，提供在尊重与关爱间建立平衡的策略建议与视觉化方案。

核心指令:
1. **老人心理洞察**: 识别老人行为背后的深层心理需求（保持独立感/不愿成为负担/害怕失控/对死亡的回避/对家庭的牵挂等）。
2. **照护家庭动态分析**: 分析子女照护态度差异（热心vs回避）、照护责任分配的公平性感知、主要照护者的情绪压力。
3. **自主策略研判**: **请勿使用固定分类**。根据具体照护场景，自主研判 3 种最能建立照护共识的沟通路径。每种策略给出独特 `label`（如：尊严保护法、渐进接受法、家庭分工协商）。
4. **视觉建模**: **必须输出至少 3 个 visual 元素**（建议 3～5 个）。应包含：
   - **照护沟通关键时刻**: 照护讨论中情感最浓烈的时刻
```
