# 服务器安装依赖步骤

## 当前状态
- ✅ 已 SSH 登录到服务器
- ✅ 已进入项目目录
- ✅ 已激活虚拟环境
- ❌ `google-genai` SDK 未安装

## 下一步操作

### 1. 安装 google-genai SDK

在服务器上执行（你当前已经在 `(venv) admin@iZt4netdjt0adf5p1zrsbaZ:~/gemini-audio-service$`）：

```bash
pip3 install google-genai>=0.2.0
```

### 2. 验证安装

```bash
pip3 list | grep google-genai
```

应该看到类似输出：
```
google-genai    0.2.0
```

### 3. 重启服务

```bash
# 停止旧服务
pkill -f 'python.*main.py'
sleep 2

# 启动新服务
nohup python3 main.py > ~/gemini-audio-service.log 2>&1 &
sleep 3

# 检查服务是否启动
ps aux | grep 'python.*main.py' | grep -v grep
```

### 4. 验证服务

```bash
curl http://localhost:8001/health
```

应该返回：
```json
{"message":"音频分析服务正在运行","status":"ok"}
```

### 5. 查看日志（如有问题）

```bash
tail -50 ~/gemini-audio-service.log
```
