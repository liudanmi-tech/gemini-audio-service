# 如何设置 Deployment Target - 超详细步骤（带截图说明）

## 🎯 你现在在哪里？

从你的截图看，你正在**代码编辑器**中查看 `Models.swift/File.swift` 文件。

要设置 Deployment Target，你需要**离开代码编辑器，进入项目设置界面**。

---

## 📍 第一步：找到项目导航器（左侧面板）

### 1. 查看 Xcode 窗口的**最左侧**

你应该能看到一个**垂直的面板**，这就是**项目导航器**（Project Navigator）。

**如果看不到左侧面板**：
- 点击顶部菜单栏的 `View` → `Navigators` → `Show Project Navigator`
- 或者按快捷键：`Command + 1`（Mac 上按 `Cmd + 1`）

### 2. 在项目导航器中，找到**最顶部**的项目名称

项目导航器通常显示为这样的结构：

```
📁 WorkSurvivalGuide (蓝色图标) ← 这个就是项目名称！
   📁 WorkSurvivalGuide
   📁 Models.swift
      📄 File.swift
   📄 NetworkManager.swift
   📄 TaskListView.swift
   ...
```

**关键点**：
- 项目名称显示为**蓝色文件夹图标** 📁
- 它位于导航器的**最顶部**
- 名称通常是 `WorkSurvivalGuide`（或你创建项目时输入的名称）

---

## 📍 第二步：点击项目名称

### 1. 在项目导航器中，**单击**最顶部的项目名称（蓝色图标）

**重要**：
- 要点击**项目名称**（蓝色图标），不是下面的文件夹
- 点击后，**中间的主编辑区域**会发生变化

### 2. 点击后，中间面板应该显示什么？

点击项目名称后，中间的主编辑区域应该显示：

```
┌─────────────────────────────────────────┐
│  General | Signing & Capabilities | ... │ ← 这些是标签
│                                          │
│  [项目信息]                              │
│                                          │
│  Deployment Info                        │
│  iOS: 17.0 ▼                            │ ← 这里！
│                                          │
│  [其他设置...]                           │
└─────────────────────────────────────────┘
```

**如果你看到这些标签**（General、Signing & Capabilities 等），说明你点击对了！

**如果还是看到代码**，说明你点击错了位置，请重新点击项目名称（蓝色图标）。

---

## 📍 第三步：点击 General 标签

### 1. 在中间面板的**顶部**，你会看到多个标签

标签包括：
- `General`（通用）
- `Signing & Capabilities`（签名和功能）
- `Resource Tags`（资源标签）
- 等等...

### 2. 点击 `General` 标签

- `General` 标签应该在第一个位置
- 点击后，标签会高亮显示（通常是蓝色或灰色背景）

---

## 📍 第四步：找到 Deployment Info

### 1. 在 `General` 标签页中，向下滚动

`Deployment Info` 部分通常在页面的**中间位置**，不在最顶部。

### 2. 查找 `Deployment Info` 标题

你应该能看到类似这样的内容：

```
┌─────────────────────────────────────┐
│  Deployment Info                    │ ← 这个标题
│  ┌───────────────────────────────┐  │
│  │ iOS: 17.0 ▼                   │  │ ← 这里显示当前版本
│  └───────────────────────────────┘  │
│                                      │
│  [其他设置...]                       │
└─────────────────────────────────────┘
```

**如果找不到 `Deployment Info`**：
- 确保你点击了 `General` 标签
- 在 `General` 标签页中，继续向下滚动
- 它可能在页面的中间或偏下位置

---

## 📍 第五步：修改 iOS 版本

### 1. 找到 `iOS` 这一行

在 `Deployment Info` 部分，第一行就是 `iOS`。

### 2. 点击版本号旁边的下拉菜单

- 版本号右侧有一个**向下的小箭头** ▼
- 点击这个小箭头，或者直接点击版本号

### 3. 选择 `16.0`

- 会弹出一个版本列表
- 在列表中，向下滚动找到 `16.0`
- 点击选择 `16.0`

### 4. 确认修改

- 选择后，版本号会自动更新为 `16.0`
- 不需要点击保存，Xcode 会自动保存

---

## ✅ 完成！

设置完成后，你应该看到：

```
Deployment Info
iOS: 16.0  ← 这里显示 16.0
```

---

## 🆘 如果还是找不到，试试这个方法

### 方法 2：通过菜单栏

1. 点击顶部菜单栏的 `File` → `Project Settings...`
2. 或者按快捷键：`Command + ,`（Mac 上按 `Cmd + ,`）
3. 在弹出的窗口中，找到 `Deployment Target`
4. 修改为 `16.0`

### 方法 3：直接搜索

1. 在 Xcode 中，按 `Command + Shift + O`（Mac 上按 `Cmd + Shift + O`）
2. 输入：`deployment`
3. 选择 `Deployment Target` 相关选项
4. 直接跳转到设置位置

## ❓ 常见问题

### Q1: 我点击了项目名称，但还是看到代码？

**A**: 可能的原因：
1. 你点击的不是项目名称，而是下面的文件夹
2. 项目名称没有正确选中

**解决方法**：
- 确保点击的是**最顶部**的蓝色图标
- 点击后，中间面板应该显示项目设置，而不是代码

### Q2: 我看到了 General 标签，但找不到 Deployment Info？

**A**: 
- 在 `General` 标签页中，**向下滚动**
- `Deployment Info` 不在最顶部，通常在中间位置
- 继续向下滚动查找

### Q3: 项目导航器在哪里？

**A**: 
- 在 Xcode 窗口的**最左侧**
- 如果看不到，按 `Cmd + 1` 打开
- 或者点击顶部菜单栏的 `View` → `Navigators` → `Show Project Navigator`
